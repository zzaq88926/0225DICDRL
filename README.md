# 個人網頁作業 - 深度強化學習課程

## 專案簡介

這是我在**國立中興大學深度強化學習課程**中完成的個人靜態網頁作業。  
本作業練習了**Git 版本控制**的基本流程，以及透過 **GitHub Pages** 部署靜態網站的方式，  
讓我對從程式碼到正式上線的完整 DevOps 流程有了更具體的認識。

## 作業目標

- 完成一個具備現代設計風格的個人介紹網頁
- 學習使用 Git 進行版本管理（`init`, `add`, `commit`, `push`）
- 透過 GitHub Pages 將網站公開上線

## 頁面功能

| 功能 | 說明 |
|------|------|
| 即時時鐘 | 使用 JavaScript `setInterval` 每秒更新，顯示當前時間與日期 |
| 個人資訊 | 顯示姓名、學校與研究方向 |
| 技能標籤 | 列出相關技術與工具 |
| GitHub 連結 | 提供外部連結至個人 GitHub 主頁 |
| 響應式設計 | 支援桌機與手機瀏覽 |

## 技術架構

- **HTML5** – 語義化結構
- **CSS3** – 玻璃態（Glassmorphism）深色主題、漸層動畫背景、`backdrop-filter`
- **Vanilla JavaScript** – 即時時鐘邏輯
- **Google Fonts (Inter)** – 現代無襯線字型
- **GitHub Pages** – 靜態網站免費託管

## 學習心得

透過本次作業，我實際走過了：

1. **撰寫前端程式碼**：理解 HTML 結構與 CSS 視覺設計的搭配方式
2. **Git 基本操作**：從 `git init` 建立本地倉庫、到 `git remote add`、`git push` 推送至遠端
3. **GitHub Pages 部署**：在 Settings → Pages 設定分支後，學會如何讓靜態網頁正式對外公開

這個過程幫助我建立了「寫程式 → 版本管理 → 部署上線」的完整技術視野，對後續課程中的實作項目很有幫助。
